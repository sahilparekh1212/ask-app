AI-Sandbox

A production-shaped backend engineering project: two Spring Boot microservices — Google OAuth2 → JWT auth and an event-driven audit trail over Kafka — with rate limiting, RBAC, a full observability stack, and a guarded LLM assistant, fronted by this Angular SPA.

Everything here runs locally with one docker compose up --build. The backend is real and hardened; this SPA is its client.

AI-Sandbox

A production-shaped Spring Boot backend — Google OAuth2 → JWT auth, an event-driven audit trail over Kafka, rate limiting, and a full observability stack — with this Angular SPA as its client.

Auth

Google sign-in and a demo login issue RSA-signed JWTs; the SPA attaches them as Bearer tokens and silently refreshes.

Audit

Every login/refresh/logout is published to Kafka and persisted; browse it as a paginated, filterable table with stats.
